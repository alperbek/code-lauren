## Run

Install node >=0.12.2

Install npm dependencies:

    $ cd path/to/project/root
    $ npm install

In a terminal:

    $ cd path/to/project/root
    $ npm run watch

In another terminal:

    $ cd path/to/project/root
    $ npm run build-pages

In another terminal:

    $ cd path/to/project/root
    $ npm run start
